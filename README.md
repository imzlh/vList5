# vlist

简单的文件管理方案<br>
使用Vue构建，与NJS集成

## 目前已经完成
 - 强大的打开方式
    - 视频
    - 音频
    - 图片
    - 二进制
    - 代码/文本
    - 字体
    - HTML类
    - ...(欢迎PR或issue)
 - 完善的UI
    - 手机端
    - PC
    - 动画(正在计划中)
 - Vue响应式设计
    - 设置
    - `getConfig`、`setConfig`
 - 文件操作
    - 复制
    - 粘贴
    - 新建
    - 上传
    - 预览操作
    - ...(欢迎PR或issue)

## 安装指南

需要NodeJS和<a href="https://github.com/imzlh/vlist-njs">NJS后端</a>

```sh
npm install
npm run build
```
